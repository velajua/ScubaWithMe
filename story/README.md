# ScubaWithMe Story Workspace

This root-level folder is the active workspace for building the ScubaWithMe story world into daily 4-frame posts, recurring arcs, character moments, and future product-friendly narrative ideas.

The working title is `Below With Me`.

## Core Story

A small reef community lives near a popular dive path. At first, the animals are scared of divers because divers can be noisy, bright, clumsy, and too close. Over time, Turtle, Grumpy Fish, Shark, and Lionfish learn to tell the difference between careless visitors and respectful observers.

The story does not teach that sea life exists to entertain humans. It teaches that if a visitor enters gently, the ocean may let them witness something real.

## Folder Map

- `00-series-overview.md`: the full premise, emotional arc, rules, and brand fit.
- `01-master-plot-points.md`: the complete plot-point bank from the chat, kept broad and expandable.
- `02-character-loop.md`: ensemble structure where each main character gets a turn as lead.
- `03-season-arcs.md`: coherent long-form arcs that can later become chapters or batches.
- `04-episode-bank.md`: 4-frame seeds, filler ideas, and repeatable post engines.
- `05-daily-workflow.md`: practical daily content workflow.
- `characters/`: numbered cast files in loop order.
- `arcs/`: flat arc overview files plus matching folders for deeper episode development.

## Naming Convention

- Root story docs use `NN-topic.md` so the reading order is obvious.
- Character files use `NN-character-name.md` in lead rotation order.
- Main arc overview files use `NN-season-arc-name.md`.
- Insert or reusable arc overview files use `NN-side-arc-name.md`.
- Matching arc folders use the same name without `.md`.
- Arc folders are extra workspace for developing finished episodes, assets, and deeper notes.
- Finished episode scripts should live inside the matching arc folder as `NN-episode-short-title.md`.

## Working Principle

Do not rush into chapters yet. First, build a rich pile of connected story material. Later, split it into tiny 4-frame groups.

## Arc Depth Rule

Each major arc should start with at least 8 episodes. Simpler arcs can stay at 8. Larger arcs can expand to 12, 16, or more once the character beats are clear.

Each episode outline should include:

- lead character
- story purpose
- 4-frame beat outline
- final turn: joke, lesson, or wonder

When an outline becomes a finished 4-frame script, keep the arc overview file and add the expanded script inside that arc's matching folder.
