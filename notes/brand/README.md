# Brand Notes

This folder holds the active ScubaWithMe brand documentation.

## Files

- `design.md`: the current brand direction, logo rules, and visual constraints.
- `assets.md`: how to use the exported Shopify brand assets and regenerate them.
- `implementation-plan.md`: the implementation record for the first brand asset pack.

## Current logo direction

- Slightly stylized but clearly anatomical sea turtle
- Nose bubbles as a small underwater accent
- `ScubaWithMe` integrated on the shell for the primary storefront mark
- Separate turtle-only icon for favicon and small avatar use

Keep this folder current when the logo direction, palette, or asset workflow changes.
