# ScubaWithMe Notes Index

This folder holds working notes for the ScubaWithMe brand, image-quality pipeline, and print-on-demand preparation.

## chaiNNer Notes

- `chainner-cli.md`: local CLI path, command pattern, and known CLI quirks.
- `chainner-image-quality-capabilities.md`: capability map for color, gradients, resize/export, cleanup, metrics, and AI upscaling.
- `chainner-final-product-templates.md`: the three current comparison templates for product-test exports.

Root helper:

- `chainner-template-helper.py`: updates template input/output paths and can run one image or a full working folder batch.
